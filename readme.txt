Dynamically display a custom uploaded or default image overlays for products in your WooCommerce store by category, status, product, inventory status, etc..

== Description ==
Now your WooCommerce products that are out of stock can be distinguished from in stock products with a dyanmically applied image overlay to all product images that are out of stock. You can use the preloaded pictures included in the plugin, or upload and set your own on the plugin settings page.
== Installation ==
Either upload a .zip file, or install through the WordPress.org repository.
